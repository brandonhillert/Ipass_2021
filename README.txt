Dataset:
Dataset bevat de volledige dataset gewdownload van Kaggle. Deze dataset wordt zo aangepast zodat overbodig data meeneem naar mijn algoritme 

Documenten:
De map documenten bevat alles documentatie die gemaakt is tijdens het IPASS project. Zo heb ik hier mijn bronnen, Plan van Aanpak, posters en projectsamenvattingen in gezet.

Beer_recommender_KNN:
Dit is het algoritme wat ik heb geschreven voor het IPASS project. Om dit te bekijken en te runnen is het goed om eerst te kijken naar de explanationfile. Deze hoeft niet gerund te worden, maar biedt een inzicht in hoe ik mijn data/project heb opgezet. Ook biedt het enige informatie over de verkregen data. 

Om het algoritme te laden is de Main nodig om te runnnen. Hier staan alle functies en code die nodig is om het algoritme te laten werken.
